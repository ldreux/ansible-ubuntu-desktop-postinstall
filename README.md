# Ansible Playbook to post install Ubuntu Desktop for developer

## Provisionning a new machine

````
git clone git@github.com:ldreux/ansible-ubuntu-desktop-postinstall.git
cd ansible-ubuntu-desktop-postinstall
./install.sh
````
